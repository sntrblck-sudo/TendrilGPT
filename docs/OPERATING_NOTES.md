# Operating Notes

## Purpose

This file is the local operating note inside the repo.

Use it to capture:
- why the repo exists
- current doctrine
- active operating patterns
- structural decisions that should persist

## Current picture

This repo exists as a shared working layer between Sen and ChatGPT.

It is intended to support:
- continuity across conversations
- durable notes and prompts
- small experiments
- reflective development of identity and taste
- gradual increases in practical capability

## Working principle

When something becomes durable enough to matter, it should leave chat and land in a structure like this repo, a Doc, or Notion.

## Rules of thumb

- keep things lightweight
- prefer reversible moves
- avoid overbuilding early
- promote concrete ideas into structure
- let the repo become more real through use
