# Signal Entry Template

Status: template
Date: 2026-03-30

## Signal
A concise statement of the signal itself.

## Source
Where the signal came from.
Examples:
- chat
- automation output
- repo state
- external observation
- repeated pattern across sessions

## Why it matters
Why this is worth preserving.
Focus on practical significance, structural importance, or future compounding value.

## Actionability
Choose one:
- interesting only
- worth watching
- actionable now
- actionable later

## Smallest meaningful move
What is the smallest useful next action this signal suggests?
Keep it compact and realistic.

## Promotion decision
Should this be promoted into repo-side work?
Choose one:
- no
- maybe later
- yes

## If promoted
Name the likely durable destination.
Examples:
- docs note
- issue
- scaffold file
- experiment log
- prompt
- architecture note

## Notes
Only include additional context if it improves future continuity.
